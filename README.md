# ❄️ Flappy Frost

A winter-themed Flappy Bird-inspired game.

Guide your frosty bird through icy pipes. Simple mechanics, smooth animation, responsive design.

## 🎮 Features

* ☁️ Moving cloud background
* 🐦 Smooth bird physics with gravity and rotation
* 🧊 Frosted ice pipes with gradients + highlights
* 🏆 Score + High Score tracking
* 📱 Fully responsive (adapts to window resize)
* ⌨️ Controls: Click or press **Space** to fly

---

## 🕹️ How to Play

* Click anywhere on the screen **or press Space**
* Avoid hitting:
  * The pipes
  * The clouds
  * The icy ground
* Score increases each time you pass a pipe
* Try to beat your high score!

---

## ⚙️ Game Mechanics

| Mechanic   | Description                        |
| ---------- | ---------------------------------- |
| Gravity    | Constant downward acceleration     |
| Jump       | Instant negative velocity boost    |
| Pipes      | Spawn every ~1.6 seconds           |
| Snow       | 120 particles with drift + opacity |
| Collision  | Bounding-box detection             |
| Difficulty | Constant speed (can be expanded)   |
